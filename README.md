# C Projects

Coleção de pequenos projetos e exercícios em **C**, feitos para praticar lógica de programação, fundamentos da linguagem e resolução de problemas.  
Cada pasta é um projeto independente, geralmente com foco em um tema específico (loops, condicionais, I/O, funções, etc).

## Como compilar e executar

### GCC (Linux / macOS / MinGW no Windows)

Entre na pasta do projeto e compile:

```bash
gcc main.c -o app
./app
```

gcc main.c -o app.exe
app.exe

## Requisitos

Um compilador C (recomendado: GCC)

Opcional: VS Code com extensão de C/C++

## Objetivo
